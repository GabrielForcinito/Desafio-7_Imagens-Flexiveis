# 🖼️ Desafio Imagens Flexíveis

## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar o uso de **imagens responsivas em HTML**, adaptando o conteúdo de acordo com o tamanho da tela do usuário.
