# 🖼️ Desafio Imagens Flexíveis

## 📌 Sobre o projeto

Projeto desenvolvido em HTML com o objetivo de praticar o uso de imagens responsivas, adaptando o conteúdo conforme o tamanho da tela do usuário.

## 🛠️ Tecnologias utilizadas

* HTML5

## 🎯 Funcionalidades

* Exibição de imagens responsivas
* Adaptação de imagens conforme resolução de tela
* Uso de recursos nativos do HTML

## 💻 Como executar

Basta abrir o arquivo `index.html` em qualquer navegador.

## 📚 Aprendizado

Neste projeto pratiquei:

* Uso de imagens responsivas
* Elementos como `<picture>` e `<source>`
* Adaptação de conteúdo para diferentes dispositivos
* Boas práticas de HTML
